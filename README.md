Third and Grove Assessment
==========

Bubble sort simulation.

### Requirements

PHP 5.6 and Composer

### Installation

1. Clone the repo

2. At the repo root:
> $ composer install

3. Install assets
> $ php app\console assets:install


4. Start the development server
> $ php app\console server:run

5. Go to local http://localhost:8000/ and enjoy :)
